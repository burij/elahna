# ⌨️ Software development 

---

Writing simple tools in Lua, self hosting with Nix and a pinch of Docker. Still learning.

Most of my public source code is currently on [GitHub](https://github.com/burij?tab=repositories&q=&type=source&language=&sort=). Here are some featured projects:

### Elahna
[elahna.burij.de](https://elahna.burij.de) is a bare-bones web server focused on (but not limited to) developing simple htmx applications in Elixir. Rewrite of Lahna.


### Lahna
[lahna.burij.de](https://lahna.burij.de) is a bare-bones web server focused on (but not limited to) developing simple htmx applications in Lua. This website is made with Lahna.

### NixOS Extended Rebuilder
A [Lua-based tool for managing NixOS](https://github.com/burij/nixos-extended-rebuilder/) configurations and system rebuilds with extended functionality.

It offers the possibility of managing additional aspects of NixOS in a declarative way without adding the complexity of flakes and Home Manager. 

### Lua Light Wings
Personal set of useful functions with beautiful syntax to extend Lua Language. Can be installed as a module via [luarocks](https://luarocks.org/modules/burij/lua-light-wings).
